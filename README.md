# Vue3.0-shop
Vue3.0 + Vuex + Vue-router + Axios + Vant 前台商城系统 <br>

该系统包含 商品首页、商品分类、首页轮播、商品详情、商品收藏、购物车、订单结算、订单详情、订单流程、个人订单管理、个人商品收藏管理、个人账号管理、个人地址管理、用户注册与用户登录等模块
